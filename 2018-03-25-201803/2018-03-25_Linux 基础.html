    Linux 最初源自 UNIX，后来有了 MINIX 再后来 linus 想把 minux 移植到 PC，牵头开发 Linux，这次有了现在的 Linux。redhat (红帽子公司)， s.u.s.e，红旗 Linux（国产）都是对 Linux 的包装。

    Linux 目录结构

    linux 的文件系统是采用层级式的树状目录结构
    /：根目录
    root：存放 root 用户的相关文件
    home：存放普通用户的相关文件
    bin：存放常用命令的目录
    sbin：存放要具有一定权限才可以使用的命令的目录
    mnt：默认挂载光驱和软驱的目录
    boot：存放引导文件的目录
    etc：存放配置相关文件
    var：存放经常变化的文件的目录
    usr：默认安装文件的文件夹

    白颜色为普通文件
    蓝颜色为文件夹
    绿颜色为可执行文件

    使用 vi 编辑器

    基本用法

    1. vi Hello.java
      public class Hello {
public static void main(String[] args){
System.out.println("HELLO WORLD");
}
}
    2. 输入i 【进入插入模式】
    3. 输入esc键【进入到命令模式】
    4. 输入: [wq表示退出保存 q!退出不保存]。即输入 :wq 或者 :q!
    5. ls 当前目录所有文件 ls -l
    6. 编译 javac Hello.java
    7. 运行 java Hello

    查找内容

    按下“/”键，这时在状态栏（也就是屏幕左下脚）就出现了 “/” 然后输入你要查找的关键字敲回车就可以了。

    找到相关文字以后：
    （1）按下小写n，向下查找
    （2）按下大写N，向上查找

    常用命令

    最高权限用户：root
    进入图形页面：startx
    关闭图形界面：init 3
    shutdown -h now 立即关机
    shutdown -r now 立即重启
    reboot 立即重启
    logout 用户注销

    init[0123456]
    0：关机
    1：单用户
    2：多用户无网络状态
    3：多用户有网络状态
    4：系统未使用保留给用户
    5：图形界面
    6：重启系统
    常用的运行级别是3和5，要修改默认运行的级别可修改文件 /etc/inittab的id:5:imitdefault: 这一行中的数字

    找回密码的方法：
    在进入 grub 引导界面时，输入 e,修改为 1，进入单用户模式。
    该方法可以找回密码，因为单用户进入是，无需输入密码，并且进去之后还可以修改密码。

    pwd 显示当前工作目录
    cd 改变目录
    ls 列出文件和目录
    ls -l
    ls -a

    mkdir 建立目录
    rmdir 删除目录

    touch 建立空文件
    cp 复制命令
    cp -r dir1 dir2 递归复制命令（复制了目录信息）

    mv 移动文件和改文件名
    rm 删除文件和目录
    rm -rf * 删除所有内容（包括目录和文件）r 递归 f 强制

    ln 建立符号连接
    ln -s 源 目标
    ln -s /etc/inittab inittab [inittab指向实际文件/etc/inittab]

    more 显示文件内容，带分页
    less 显示文件内容带分页
    grep 在文本中查询内容[grep -n “shunping” aaa.java]
    | 管道命令[把上一个命令的结果交给 | 的后面的命令处理]
    man 相当于 help 命令

    find / -name aaa.java 从根目录按照文件名称查找 aaa.java 文件

    ls -l > a.txt 列表的内容写入文件a.txt中（覆盖写）
    ls -al >> aa.txt 列表内容追加到文件aa.txt的末尾
    database_program < database_data 从文件中输入信息

    用户和组

    添加组
    groupadd policeman
    cat /etc/group | more
    groupadd tufei

    创建用户，并指定组名
    useradd -g policeman along
    useradd -g policeman panpan
    useradd -g tufei ashuai

    cat /etc/passwd
    ashuai:x:502:501::/home/ashuai:/bin/bash
    用户名：密码：用户ID：用户所在组ID:注释：用户的主目录：用户使用的sshell解析器

    -rw-r–r–
    -：文件类型（-：文件，d：目录，l:连接）
    rw-：文件的所有者对该文件的权限，r可读（4），w可写（4），可执行（1）
    r–：文件所在组对该文件的权限
    r–：其他组的用户对该文件的权限

    修改文件访问权限
    chmod 777 along

    修改用户所在组
    usemod -g policeman ashuai

    chown 修改文件的所有组

    df /hpme :挂载目录。sda3：a 表示第一个，3表示第三块分区
