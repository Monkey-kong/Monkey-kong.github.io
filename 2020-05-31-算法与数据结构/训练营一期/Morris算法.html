    算法基础之 Morris

    1. 算法规则

    1. cur 无左子树， cur = cur.right（当前节点右移）
    2. cur 有左子树，找到左子树最右节点，most right
      1. 如果 most right 的右指针指向 null。most right.right = cur, cur = cur.left（最右节点右指针指向当前节点，当前节点左移）
      2. 如果 most right 的右指针指向当前节点（cur）。most right.right=null,cur=cur.right（最右节点右指针指向 null，当前节点右移）

    例如有如下二叉树(红色 cur 表示当前节点，绿色 cur 表示下一个 cur 节点)：

    所以生成的 Morris 序列为： 1,2,4,2,5,1,3,6,3,7

    例图2：

    可以发现有如下特点：

    • 有左子树的节点肯定会碰到两次
    • 当当前节点的左子树最右节点的右指针指向自己时，表示时第二次碰到了
    • 时间复杂度为 O(N)，因为每个节点最多遍历两次

    2. Morris 算法代码实现

    public class Morris {
    public static void main(String[] args) {
        Node d = new Node(1);
        d.left = new Node(2);
        d.right = new Node(3);
        d.left.left = new Node(4);
        d.left.right = new Node(5);
        d.right.left = new Node(6);
        d.right.right = new Node(7);
        morris(d);
    }

    public static class Node {
        public int value;
        public Node left;
        public Node right;
        public Node(int v) {
            value = v;
        }
    }

    private static void morris(Node head) {
        if (head == null) return;
        Node cur = head;
        Node mostRight = null;
        while (cur != null) {
            System.out.println(cur.value);
            // 左子树为空,cur 右移
            if (cur.left == null) {
                cur = cur.right;
            } else {
                // 查找左子树最右节点
                mostRight = cur.left;
                while (mostRight.right != null && mostRight.right != cur) {
                    mostRight = mostRight.right;
                }
                // 最右节点右指针为 null
                if (mostRight.right == null) {
                    mostRight.right = cur;
                    cur = cur.left;
                } else { // 最右节点右指针为 cur
                    mostRight.right = null;
                    cur = cur.right;
                }
            }
        }
    }
}

    3. 使用 Morris 实现前、中、后序遍历

    先序遍历

    • 访问根节点
    • 左子树
    • 右子树

    实现：有左子树的节点，第二次访问时不打印即可

    中序遍历

    • 左子树
    • 根节点
    • 右子树

    实现：有左子树的节点，第一次访问时不打印即可

    后序遍历

    • 左子树
    • 右子树
    • 根节点

    实现：有左子树的节点，第二次访问时，逆序打印左子树的右边界，最后单独逆序打印整树的右边界即可。

    // 最右节点右指针为 null
if (mostRight.right == null) {
    mostRight.right = cur;
    cur = cur.left;
} else { // 最右节点右指针为 cur
    mostRight.right = null;
    // 这里注意需要在最右节点右指针设置为 null 后再打印
    printRightBorderRev(cur.left);
    cur = cur.right;
}

// 反转左子树右边界输出后还原
private static void printRightBorderRev(Node node) {
    if (node == null) return;
    // 反转列表
    Node pre = reverseNode(node);
    Node preBak = pre;
    // 遍历输出
    while (pre != null) {
        System.out.print(pre.value + ",");
        pre = pre.right;
    }
    node = reverseNode(preBak);
    int i = 10;
}

private static Node reverseNode(Node node) {
    Node pre = null;
    Node cur = node;
    while (cur != null) {
        Node tmp = cur.right;
        cur.right = pre;
        pre = cur;
        cur = tmp;
    }
    return pre;
}

    4. 验证是否为搜索二叉树

    线索二叉树：每个节点左节点小于根节点，右节点大于跟节点。

    思路：

    • 通常的验证流程就是中序遍历，验证每个节点的值比前一个大即可。

    实现：

    • 中序遍历打印的地方修改为判断即可

    5. 二叉树最小高度

    5.1 使用 Morris 实现思路

    • 维护一个变量用于保存最小高度
    • 记录当前节点的高度
      • cur 无左树，cur 右移动，直接 level++
      • cur 有左树
        • 第一次访问：cur 左移，也是直接 level++
        • 第二次访问：cur 右移，level = level - 左子树右边界的高度
    • 如果当前节点为叶子节点，尝试更新最小高度
      • cur 有左数，并且第二次访问时，如果前面节点左指针为 null，说明为叶子节点。
    • 最后单独计算整棵树的最右节点是否为叶子节点，如果是则尝试更新最小高度

    5.2 代码

    private static int minHeight(Node head) {
    if (head == null) return 0;

    Node cur = head;
    Node mostRight = null;
    int minHeight = Integer.MAX_VALUE;
    int level = 0;
    while (cur != null) {
        // 左子树为空,cur 右移
        if (cur.left == null) {
            cur = cur.right;
            level++;
        } else {
            // 查找左子树最右节点
            mostRight = cur.left;
            // 左子树有边界
            int rightBorder = 1;
            while (mostRight.right != null && mostRight.right != cur) {
                mostRight = mostRight.right;
                rightBorder++;
            }
            // 最右节点右指针为 null
            if (mostRight.right == null) {
                mostRight.right = cur;
                level++;
                cur = cur.left;
            } else { // 最右节点右指针为 cur
                if (mostRight.left == null) {
                    // 此时的 level 就是叶子节点的 level
                    minHeight = Math.min(minHeight, level);
                }
                level -= rightBorder;
                mostRight.right = null;
                cur = cur.right;
            }
        }
    }

    // 单独计算整数最右节点
    Node t = head;
    int fullRightBorder = 1;
    while (t.right != null) {
        fullRightBorder++;
        t = t.right;
    }
    if (t.left == null) {
        minHeight = Math.min(minHeight, fullRightBorder);
    }

    return minHeight;
}
