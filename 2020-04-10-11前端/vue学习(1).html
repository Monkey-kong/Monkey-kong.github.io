    1.2 v-text 更新标签文本内容

    简写：双大括号

    v-html：和 v-text 类似。如果渲染数据为标签形式字符串，会真的渲染为标签。

    <html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <script src="./js/vue.js"></script>
</head>
<body>
    <div id="app">
        <h1>下面是指令操作文本</h1>
        <p v-text="msg"></p>
        <hr>
        <!-- v-text 的简写 -->
        <p>{{msg}}</p>
        <!-- 如果是标签形式，直接渲染 -->
        <p v-html="msg"></p>
    </div>
</body>
</html>
<script>
    var vue = new Vue({
        el: "#app",
        data: {
            msg: "<a href='http://www/baidu.com'>百度</a>"
        }
    });
</script>

    </div>

    v-for 小练习：

    同时学习 vue 中 v-model 用法。

    <body>
    <div id="app">
        <p>
            <!-- v-model -->
            <input type="text" v-model="hobby">
            <button @click="add">单击我添加爱好</button>
        </p>
        <ul>
            <li v-for="item in arr">{{item}}</li>
        </ul>
    </div>
</body>
<script>
    var vm = new Vue({
        el: "#app",
        data: {
            arr: ['吃饭', '睡觉', '打豆豆'],
            hobby: ''
        },
        methods: {
            add() {
                // 向数组尾部添加元素
                vm.arr.push(this.hobby);
                // 清空输入框
                vm.hobby = '';
            }
        }
    });
</script>

    1.6 v-on 绑定事件

    缩写：@

    在普通元素上时，只能监听原生 DOM 事件。 用在自定义元素组件上时，也可以监听子组件触发的自定义事件。 

    <body>
    <div id="app">
        <h1>我是计数器小案例{{count}}</h1>
        <button v-on:click="add">单击我+1</button>
        <button @click="minus">单击我-1</button>
        <p @mouseenter="haha">我绑定鼠标移上事件</p>
    </div>
</body>
<script>
    var vm = new Vue({
        el: "#app",
        data: {
            count: 1
        },
        methods: {
            add() {
                this.count++;
                // 设置上限
                this.count = this.count >=10 ? 10 : this.count;
            },
            minus() {
                this.count--;
                this.count = this.count <= -6 ? -1000000 : this.count;
            },
            haha() {
                alert("鼠标在我上面");
            }
        }
    });
</script>

    1.7 v-model 数据双向绑定

    只有当实例被创建时就已经存在于 data 中的 property 才是响应式的。

    只能给表单元素使用。给非表单元素使用后没有任何效果。

    <body>
    <div id="app">
        <h1>我是实例属性值{{msg}}</h1>
        <input type="text" v-model="msg">
    </div>
</body>
<script>
    var vm = new Vue({
        el: "#app",
        data: {
            msg: "我是实例数据"
        }
    });
</script>

    注意：v-model 一般用于表单元素，实现数据双向绑定。v-bind 可以绑定 class、style、attr 等，是单向的绑定。

    2.4 animation

    概述:animation是CSS3中新增的动画样式，它的使用类似于函数，分为两步：声明和调用。

    Animation这个属性可以让程序员在样式当中完成动画效果。

    注意：以前让元素运动一般都是需要通过JS【比如：JQ】

    浏览器私有前缀：

    谷歌 微软 火狐 欧鹏
    webkit ms Moz o 
    <html lang="en">
<head>

    <meta name="viewport" content="width=device-width, initial-scale=1.0">

        * {
            margin: 0;
            padding: 0;
        }
        div {
            position: relative;
            width: 300px;
            height: 150px;
            background-color: cyan;
            /* 调用动画 */
            /* 
              第一个参数：动画名称
              第二个参数：每一次动画执行时间
              第三个参数：动画执行速率
              第四个参数：第一次动画延迟时间

    2.5 watch 监听实例属性

    <body>

    </div>

<script>
    var vm = new Vue({
        el: "#app",
        data: {
            msg: ""
        },
        // 监听实例属性值变化地方
        watch: {
            // watch 可以监听实例属性值变化
            // 这里书写函数名字务必和监听属性的名字相同
            msg(a) {
                console.log(a);
                console.log("你发生变化了");
            }
        }

    3. vue 声明周期函数

    概述：在 Vue 实例和模板 DOM 标签进行关联的时候，其实这个阶段经历很多种状态，

    可以通过Vue框架提供生命周期函数，可以在不同的状态下，书写自己业务代码即可。

    注意：在特定的条件下执行函数而已。（面试常问道：八个）

    <body>
    <div id="app">
        <h1>生命周期函数-------{{msg}}</h1>
        <button @click="add">单击我我加上1</button>
    </div>
</body>
<script>
    var vm = new Vue({
        el: "#app",
        data: {
            msg: 1
        },
        methods: {
            add() {
                this.msg++;
            }
        },
        watch: {},
        // 生命周期函数
        beforeCreate() {
            // 在创建实例之前执行一次，这里访问不到实例的 data 数据
            console.log("beforeCreate 在创建 vm 之前执行一次" + this.msg);
        },
        created() {
            // 在实例创建完毕后立即执行一次
            console.log("created 在创建 vm 后立即执行一次" + this.msg)
        },
        beforeMount() {
            // 在实例创建完毕且和模板进行关联之前执行一次
            console.log("beforeMount 我是 vm 即将和模板进行关联" + this.msg)
        },
        mounted() {
            // 在实际工作中经常从这个生命周期函数拉取服务器数据
            console.log("mounted 我是 vm 我已经和模板关联完毕" + this.msg);
        },
        beforeUpdate() {
            console.log("beforeUpdate 当数据发生变化时立即执行一次" + this.msg);
        },
        updated() {
            console.log("updated 当数据发生变化后立即执行一次" + this.msg)
        }
    });
</script>
