记录使用 VMWare 安装 Linux 系统、配置网络，然后克隆多台主机的简要步骤，防止忘记。

Windows 10 安装 Ubuntu 子系统。

1. 系统安装

  1. 安装 VMWare
  2. 新建空的虚拟机
  3. 配置 CD/DVD，执行 iso 文件
  4. 开启虚拟机安装系统
  5. 分三个区：boot 分区、swap 分区（内存的两倍）、用户分区

2. 配置网络

3. 克隆虚拟机

  1. 先给 node01 打个快照，然后克隆

  2. 进入系统，修改 ip 地址

  3. 修改 hostname: cd /etc/sysconfig vi network

  4. 删除文件：rm -f /etc/udev/rules.d/70-presistent-net.rules

  5. 重启：reboot

4. Windows 10 安装 Ubuntu 子系统

  1. 设置==>更新和安全==>开发者选项==>选中【开发人员模式】
  2. win+s==>搜索【启用或关闭 Windows 功能】==> 选中 【适用于 Linux 的 Windows 子系统】
  3. 应用商店安装 Ubuntu
  4. 设置密码和修改配置
