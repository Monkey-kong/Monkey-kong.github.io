    public static Node reverseLinkedList(Node h) {
    Node pre = null;
    Node next = null;
    while (h != null) {
        next = h.next;
        h.next = pre;
        pre = h;
        h = next;
    }
    return pre;
}

    public static Node testReverseLinkedList(Node h) {
    if (h == null) {
        return null;
    }
    // 把所有 node 装入 list
    ArrayList<Node> nodes = new ArrayList<>();
    while (h != null) {
        nodes.add(h);
        h = h.next;
    }
    // 重新设置每个 node 的 next 指针
    nodes.get(0).next = null;
    for (int i = 1; i < nodes.size(); i++) {
        nodes.get(i).next = nodes.get(i-1);
    }
    // 返回最后一个节点就是新链表的头节点
    return nodes.get(nodes.size() - 1);
}


public static DoubleNode testReverseDoubleList(DoubleNode h) {
    if (h == null) {
        return null;
    }
    // 把所有 node 装入 list
    ArrayList<DoubleNode> nodes = new ArrayList<>();
    while (h != null) {
        nodes.add(h);
        h = h.next;

    // 重新设置每个 node 的 next 指针和 pre 指针
    nodes.get(0).next = null;
    for (int i = 1; i < nodes.size(); i++) {
        nodes.get(i-1).pre = nodes.get(i);
        nodes.get(i).next = nodes.get(i-1);
    }

    // 返回最后一个节点就是新链表的头节点
    return nodes.get(nodes.size() - 1);
}

    把给定值都删除

    public static Node removeValue(Node h, int num) {
    // 找到新头部
    while (h != null) {
        if (h.value != num) {
            break;
        }
        h = h.next;
    }
    if (h == null) {
        return null;
    }
    // head来到 第一个不需要删的位置
    Node pre = h;
    Node cur = h;
    while (cur.next != null) {
        cur = cur.next;
        if (cur.value == num) {
            pre.next = cur.next;
        } else {
            pre = cur;
        }
    }
    return h;
}

    2、栈和队列

    2.1 逻辑概念

    • 栈：数据先进后出，犹如弹匣
    • 队列：数据先进先出，好似排队

    2.2 栈和队列实际实现

    双向链表实现

    四个基础方法全部实现，自由组合即可。

    双向链表，维护 head、tail 两个指针的移动。所有操作均是 O(1)。

    public static class Node<T> {
    public T value;
    public Node<T> next;
    public Node<T> pre;
    public Node(T value) {
        this.value = value;
    }
}

public static class DoubleEndsQueue<T> {
    Node<T> head;
    Node<T> tail;

    public void addFromHead(T value) {
        Node<T> cur = new Node<>(value);
        if (head == null) {
            head = cur;
            tail = cur;
        } else {
            cur.next = head;
            head.pre = cur;
            head = cur;
        }
    }

    public void addFromTail(T value) {
        Node<T> cur = new Node<>(value);
        if (head == null) {
            head = cur;
            tail = cur;
        } else {
            cur.pre = tail;
            tail.next = cur;
            tail = cur;
        }
    }

    public T popFromHead() {
        if (head == null) {
            return null;
        }
        Node<T> cur = head;
        if (head == tail) {
            head = null;
            tail = null;
        } else {
            head = head.next;
            head.pre = null;
            cur.next = null;
        }
        return cur.value;
    }

    public T popFromTail() {
        if (head == null) {
            return null;
        }
        Node<T> cur = tail;
        if (head == tail) {
            head = null;
            tail = null;
        } else {
            tail = tail.pre;
            tail.next = null;
            cur.pre = null;
        }
        return cur.value;
    }

    public boolean isEmpty() {
        return head == null;
    }
}

    数组实现（固定大小）

    数组实现固定大小栈

    public static class MyStack {
    private int[] arr;
    // 下一个入栈位置。也就是下一个出栈前一个位置
    private int index;
    private final int limit;

    public MyStack(int limit) {
        this.limit = limit;
        index = 0;
        arr = new int[limit];
    }

    public void push(int num) {
        if (index >= limit) {
            throw new RuntimeException("栈已满，不能再加！");
        }
        arr[index] = num;
        index++;
    }

    public int pop() {
        if (index == 0) {
            throw new RuntimeException("栈为空，不能再拿！");
        }
        int res = arr[index-1];
        index--;
        return res;
    }
}

    环形数组实现固定大小队列

    public static class MyQueue {
    // 固定大小数组
    private int[] arr;
    // 下一个入队位置
    private int pushi;
    // 下一个出队位置
    private int popi;
    // 最大大小
    private final int limit;
    // 当前队列大小
    private int size;

    public MyQueue(int limit) {
        pushi = 0;
        popi = 0;
        size = 0;
        this.limit = limit;
        arr = new int[limit];
    }

    public void push(int num) {
        if (limit == size) {
            throw new RuntimeException("队列已满，不能再加！");
        }
        arr[pushi] = num;
        size++;
        pushi = nextIndex(pushi);
    }

    public int pop() {
        if (size == 0) {
            throw new RuntimeException("队列为空，不能再拿！");
        }
        int res = arr[popi];
        size--;
        popi = nextIndex(popi);
        return res;
    }

    public boolean isEmpty() {
        return size == 0;
    }

    // 如果现在的下标是i，返回下一个位置
    private int nextIndex(int index) {
        // 入队的顺序 012345012345
        // 出队的顺序 012345012345
        return (index < limit -1) ? index+1 : 0;
    }
}

    2、实现一个特俗的栈，在基本功能的基础上，再实现返回栈中最小元素的功能

    • 要求 pop、push、getMin 都是 O(1)
    • 可以使用现有栈结构

    思路：弄两个栈。一个数据栈一个最小栈，两个栈等高。

    public static class MyStack {
    private Stack<Integer> dataStack;
    private Stack<Integer> minStack;

    public MyStack() {
        dataStack = new Stack<>();
        minStack = new Stack<>();
    }

    public void push(int num) {
        if (minStack.isEmpty()) {
            minStack.push(num);
        } else {
            // 最小栈栈顶和num比较
            minStack.push(Math.min(minStack.peek(), num));
        }

        dataStack.push(num);
    }

    public int pop() {
        if (minStack.isEmpty()) {
            throw new RuntimeException("栈为空，不可拿数据！");
        }
        minStack.pop();
        return dataStack.pop();
    }

    public int getMin() {
        if (this.minStack.isEmpty()) {
            throw new RuntimeException("栈为空，不可拿数据！");
        }
        return minStack.peek();
    }
}

    3、用栈结构实现队列结构

    public static class MyQueue {
    private Stack<Integer> pushStack;
    private Stack<Integer> pollStack;

    public MyQueue() {
        pushStack = new Stack<>();
        pollStack = new Stack<>();
    }

    public void push(Integer num) {
        pushStack.push(num);
        pushToPoll();
    }

    public Integer poll() {
        if (isEmpty()) {
            throw new RuntimeException("队列为空，不能再拿数据");
        }
        pushToPoll();
        return pollStack.pop();
    }

    public Integer peek() {
        if (isEmpty()) {
            throw new RuntimeException("队列为空，不能再拿数据");
        }
        pushToPoll();
        return pollStack.peek();
    }

    public boolean isEmpty() {
        return pushStack.isEmpty() && pollStack.isEmpty();
    }

    private void pushToPoll() {
        if (pollStack.isEmpty()) {
            // 一定要遍历完
            while (!pushStack.isEmpty()) {
                pollStack.push(pushStack.pop());
            }
        }
    }
}

    4、用队列结构实现栈结构

    public static class MyStack {
    private Queue<Integer> queue;
    private Queue<Integer> help;
    public MyStack() {
        queue = new LinkedList<>();
        help = new LinkedList<>();
    }

    public void push(Integer num) {
        queue.offer(num);
    }

    public Integer pop() {
        queueToHelp();
        Integer res = queue.poll();
        helpToQueue();
        return res;
    }

    public Integer peek() {
        queueToHelp();
        Integer res = queue.poll();
        help.offer(res);
        helpToQueue();
        return res;
    }

    public boolean isEmpty() {
        return queue.isEmpty();
    }

    private void queueToHelp() {
        while (queue.size() > 1) {
            help.offer(queue.poll());
        }
    }

    private void helpToQueue() {
        Queue<Integer> tmp = queue;
        queue = help;
        help = tmp;
    }
}

    3、递归

    3.1 递归例子

    求数组arr[L..R]中的最大值，怎么用递归方法实现。

    1）将[L..R]范围分成左右两半。左：[L..Mid] 右[Mid+1..R]

    2）左部分求最大值，右部分求最大值

    3） [L..R]范围上的最大值，是 max{左部分最大值，右部分最大值}

    注意：2）是个递归过程，当范围上只有一个数，就可以不用再递归了

    public static Integer getMax(int[] arr) {
    if (arr == null || arr.length == 0) {
        return 0;
    }
    return process(arr, 0, arr.length - 1);
}
private static Integer process(int[] arr, int L, int R) {
    if (L == R) { // arr[L..R]范围上只有一个数，直接返回，base case
        return arr[L];
    }
    int mid = L + ((R - L) >> 1);
    Integer l = process(arr, L, mid);
    Integer r = process(arr, mid + 1, R);
    return Math.max(l, r);
}

    3.2 递归的脑图

    递归的脑图有利于理解和分析递归，递归不是玄学，递归底层是利用系统栈来实现的。任何递归函数都一定可以改成非递归。

    3.3 Master 公式

    形如 *T(N) = a * T(N/b) + O(N^d)(其中的a、b、d都是常数)*的递归函数，可以直接通过Master公式来确定时间复杂度

    • 如果 log(b,a) < d，复杂度为O(N^d)

    • 如果 log(b,a) > d，复杂度为O(N^log(b,a))

    • 如果 log(b,a) == d，复杂度为O(N^d * logN)

    上面例子：T(N) = 2 * T(N/2) + O(N^0)。所以 log(2,2) > 0，所以复杂度为 O(N^log(2,2)) = O(N)。

    4、哈希表和有序表

    4.1 哈希表

    1. 在使用层面上可以理解为一种集合结构

    2. 如果只有 key，没有伴随数据 value，可以使用 HashSet 结构

    3. 如果既有 key，又有伴随数据 value，可以使用 HashMap 结构

    4. 有无伴随数据是 HashMap 和 HashSet 唯一的区别，底层的实际结构是一回事

    5. 使用哈希表增（put）、删（remove）、改（put）和查（get）操作可以认为时间复杂度为 O(1)，但是常数时间比较大

    6. 放入哈希表的东西，如果是基础类型，内部按值传递，内存占用是这个东西的大小

    7. 放入哈希表的东西，如果是引用类型，内部按引用传递，内存占用是 8 字节

    4.2 有序表

    1. 有序表在使用层面上可以理解为一种集合结构
    2. 有 key 无 value，可以使用 TreeSet
    3. 有 key 有 value，可以使用 TreeMap
    4. 有无 value 是 TreeSet 和 TreeMap 的唯一区别，底层结构一样
    5. 使用有序表增（put）、删（remove）、改（put）和查（get）操作可以认为时间复杂度为 O(logN)
    6. 放基础类型，按值传递，内存占用同值大小相同
    7. 放引用类型，按引用传递，内存占用8字节
    8. 有序表把 key 按照顺序组织起来，而哈希表完全不组织
    9. 红黑树、AVL树、size-balance-tree和跳表等都属于有序表结构，只是底层具体实现不同
    10. 不管是什么底层具体实现，只要是有序表，都有以下固定的基本功能和固定的时间复杂度 O(logN)
      • void put(K key, V value)，向有序表添加或更新记录
      • V get(K key)，根据 key 查询 value
      • void remove(K key)
      • boolean containsKey(K key)
      • K firstKey()，返回所有键值的排序结果中，最小的那个
      • K lastKey()，返回最大那个 key
      • K floorKey(K key)，返回 <= key，离 key 最近的那个
      • K ceilingKey(K key)，返回 >= key，离 key 最近的那个
